OVERVIEW

Lazius (lot-zee-us) is a browser-based inventory management program designed for rapid data entry. Adding items to the inventory does not require a mouse, the Enter key moves to the next item. This allows a large number of items to be entered into the inventory very quickly, without the hassle of clicking between different input fields in a form. Editing entries and searching the inventory are also performed with the keyboard.

Data is stored in the JSON format, using the browser's local storage. The program does not rely on any prior libraries or frameworks. Data can be copied to the clipboard and pasted to a text file for long-term storage.

PROGRAM LAYOUT

The Lazius database uses a database object with the following structure:

{
"name":"DB Test",
"comments":"",
"contactInfo":"",
"dataStructureArray":[
{
"parameterName":"title",
"parameterToDisplay":"Title",
"parameterNotes":"",
"parameterInputType":"input",
"parameterAutoComplete":"no"},
{
"parameterName":"authorLastName",
"parameterToDisplay":"Author, Last Name",
"parameterNotes":"",
"parameterInputType":"input",
"parameterAutoComplete":"no"
},
{
"parameterName":"authorFirstName", "parameterToDisplay":"Author, First Name", "parameterNotes":"", "parameterInputType":"input", "parameterAutoComplete":"no" }, { "parameterName":"location", "parameterToDisplay":"Location", "parameterNotes":"", "parameterInputType":"input", "parameterAutoComplete":"yes" }, { "parameterName":"topics", "parameterToDisplay":"Topic", "parameterNotes":"", "parameterInputType":"array", "parameterAutoComplete":"no" }, { "parameterName":"quantity", "parameterToDisplay":"Quantity", "parameterNotes":"", "parameterInputType":"number", "parameterAutoComplete":"no" }, { "parameterName":"id", "parameterToDisplay":"ID", "parameterNotes":"", "parameterInputType":"number", "parameterAutoComplete":"no" }], "databaseArray":[ { "title":"Armchair Universe", "authorLastName":"Dewdney", "authorFirstName":"A.K.", "location":"Main Shelf", "topics":["fractals","computer science","recreational mathematics","analog computers"], "quantity":1, "id":1 }, { "title":"Kitty and the Midnight Hour", "authorLastName":"Vaughn", "authorFirstName":"Carrie", "location":"Main Shelf", "topics":["Horror","Romance Horror","Talk Radio","Werewolves","Vampires"], "quantity":1, "id":2 }], "bufferArray":[], "dateCreated":"2020-02-05T17:26:43.869Z" }
  • The Data Structure Array contains all information on the database parameters: their names, how those names are displayed to the user, and what kind of information the parameter uses (either a string, number, or array)
  • The Database Array contains all the contents of the database.
  • The Buffer Array is used to store values before they are applied to the database.

This object is stored in the browser's local storage. Local storage allocates at least 5 MB per domain, which is plenty for small-scale databases.
